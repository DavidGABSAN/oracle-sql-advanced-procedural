# Oracle SQL Advanced & Procedural Programming 🏛️💾

Repositorio orientado a la implementación de esquemas relacionales complejos y lógica de negocio mediante PL/SQL. Demuestra el dominio en la automatización de integridad de datos y consultas avanzadas.

## ⚙️ Competencias Avanzadas
* **Automatización de Integridad (Triggers):** Implementación de disparadores (`AFTER INSERT OR DELETE`) para mantener la sincronización de datos derivados, garantizando que el estado del sistema sea siempre coherente sin intervención manual.
* **Programación Procedimental (PL/SQL):** * Creación de funciones que devuelven cursores dinámicos (`SYS_REFCURSOR`) para la explotación de datos complejos.
    * Lógica de filtrado avanzado utilizando `NOT EXISTS` para resolución de conflictos en rangos temporales (ideal para sistemas de reservas).
* **Modelado y Normalización:** Diseño de esquemas (ER) aplicando restricciones de integridad (`UNIQUE`, `NOT NULL`, `CHECK`) y claves foráneas, equilibrando la teoría de la 3FN con la optimización de rendimiento.

## 📊 Casos de Uso Destacados
* **Gestión Deportiva:** Sistema de contabilidad automática de puntos en partidos de baloncesto mediante triggers de eventos de enceste.
* **Sistema de Reservas:** Motor de búsqueda de disponibilidad hotelera basado en rangos de fechas, evitando solapamientos mediante subconsultas analíticas.
